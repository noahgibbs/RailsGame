RailsGame

Installing

First, install the RailsGame gem: "sudo gem install noahgibbs-railsgame --source http://gems.github.com".

You'll also need a Rails project. Type "rails myproject" in the correct location, just as you always would.

Generate crypto keys for your project: "./script/generate rg_crypto_keys" will do it. [NOTE: need to add something to handle environment variable when this is run, just as ./run_server.sh does.]

Soon: generate a RailsGame view: "./script/generate rg_scaffold <name>" where name is the name of the controller.
